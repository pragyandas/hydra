## Intro

This chat application demonstrates Hydra, a distributed virtual actor system built on NATS JetStream. Actors handle messages, manage state, and enable real-time communication between users through a WebSocket frontend.

## Getting started

```sh
go run .
```
Open http://localhost:8080
